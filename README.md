# flutter-web-android-animated-random-string

## Description
A POC project for flutter with animation.

## Tech stack
- flutter
- android

## Docker stack
- matspfeiffer/flutter:beta

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
